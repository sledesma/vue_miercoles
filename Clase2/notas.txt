ESTRUCTURA DE ARCHIVOS Y CARPETAS
1) Dependencias (node_modules) - NO SE TOCA
2) Archivos estáticos (public)
3) Archivos fuente (src)
4) Archivos de configuración (todos los archivos sueltos en el dir raíz)

dist: EL RESULTADO DEL DESARROLLO


ETAPAS DEL DESARROLLO
vue create (Crear el proyecto)
npm run serve (Desarrollar el proyecto)
npm run build (Publicar el proyecto)



PROCESO PARA DESARROLLAR EN vue

1) Empezar con un mock
2) Dividir el mock en componentes
3) Crear una versión estática
4) Definir la info de la página
5) Definir donde debe vivir dicha info
6) Definir qué cambia cuando la info cambia
